*ngClass for class binding.

ngClass binding allows multiple classes to be bound with an element with only one class binding event.
When using class binding on an element we previously had to code each class binding separately.

ngClass is an attribute directive. It is used to modify an attribute on an existing DOM element.

isActive: {{ isActive }}

<p class="class"
[class.active] = "isActive"
[class.not-active] = "!isActive">
</p>

<p class="class"
[ngClass]="{
active: isActive,
'not-active': !isActive
}"
</p>
