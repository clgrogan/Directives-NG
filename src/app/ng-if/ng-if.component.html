*ngIf

div using '*ngIf="courses.length > 0"'' is truthy so there are courses.

d div using '*ngIf="courses.length === 0'' is truthy. No courses exist.

ng-template

div *ngIf="courses.length > 0; then coursesList; else noCourses" condition is truthy so courses exist #coursesList.

div *ngIf="courses.length > 0; then coursesList; else noCourses" condition is falsy so NO courses exist #noCourses.

hidden element property with conditional

div [hidden]="courses.length === 0". If the condition met, then the element is NOT hidden.

div [hidden]="courses.length === 0". If the condition is NOT met, then the element is hidden.
