*ngStyle for style binding

ngStyle binding allows multiple styles to be bound with an element with only one class binding event.
Using *ngStyle is not a best practice.
toggle canSave value: {{ canSave }}

<button
[style.backgroundColor]="canSave ? 'blue' : 'gray'"
[style.color]="canSave ? 'white' : 'black'"
[style.fontWeight]="canSave ? 'bold' : 'normal'"
(click)="onClick()">
Save </button>

toggle canSave value: {{ canSave }}

<button
[ngStyle]="{
backgroundColor: canSave ? 'blue' : 'gray',
color: canSave ? 'white' : 'black',
fontWeight: canSave ? 'bold' : 'normal'
}"
(click)="onClick()">
Save </button>
